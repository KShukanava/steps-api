# @qavajs/api-steps
This is a @qavajs package to get basic API cucumber steps.

## Installation

`npm install @qavajs/steps-api`

## Configuration
```javascript
module.exports = {
    default: {
        require: ['@qavajs/steps-api']
    }
}
```
